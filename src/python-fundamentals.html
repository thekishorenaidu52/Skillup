Python Fundamentals

1. Basic Python Syntax

Python is known for its clean and readable syntax. Here are the fundamental concepts:

Variables and Data Types

# Variables and basic data types name = "John" # String age = 25 # Integer height = 1.75 # Float is_student = True # Boolean # Type checking print(type(name)) # print(type(age)) # # String operations full_name = "John" + " " + "Doe" # Concatenation print(name.upper()) # Convert to uppercase print(len(name)) # String length

Basic Operations

# Arithmetic operations x = 10 y = 3 print(x + y) # Addition: 13 print(x - y) # Subtraction: 7 print(x * y) # Multiplication: 30 print(x / y) # Division: 3.333... print(x // y) # Floor division: 3 print(x % y) # Modulus: 1 print(x ** y) # Exponentiation: 1000

2. Control Flow

Conditional Statements

# If-elif-else statements score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") else: print("Grade: D")

Loops

# For loop for i in range(5): print(i) # Prints 0 to 4 # While loop count = 0 while count < 5: print(count) count += 1 # Loop control for num in range(10): if num == 5: continue # Skip 5 if num == 8: break # Stop at 8 print(num)

3. Data Structures

Lists

# List operations fruits = ["apple", "banana", "orange"] fruits.append("grape") # Add item fruits.remove("banana") # Remove item fruits.insert(1, "mango") # Insert at position print(fruits[0]) # Access item print(fruits[-1]) # Last item print(fruits[1:3]) # Slicing

Dictionaries

# Dictionary operations person = { "name": "John", "age": 25, "city": "New York" } print(person["name"]) # Access value person["email"] = "john@example.com" # Add key-value del person["age"] # Remove key-value print(person.keys()) # Get all keys print(person.values()) # Get all values
Tip: Dictionaries are useful for storing key-value pairs and are commonly used in data processing.

4. Functions

# Basic function def greet(name): return f"Hello, {name}!" # Function with default parameter def power(base, exponent=2): return base ** exponent # Lambda function square = lambda x: x**2 # Function with multiple returns def get_stats(numbers): return min(numbers), max(numbers), sum(numbers)/len(numbers) # Using the function numbers = [1, 2, 3, 4, 5] minimum, maximum, average = get_stats(numbers)

5. Error Handling

# Try-except block try: number = int(input("Enter a number: ")) result = 10 / number print(result) except ValueError: print("Please enter a valid number") except ZeroDivisionError: print("Cannot divide by zero") finally: print("Execution completed")
Best Practice: Always use try-except blocks when dealing with potential errors in your code.

6. File Handling

# Reading a file with open('data.txt', 'r') as file: content = file.read() lines = file.readlines() # Writing to a file with open('output.txt', 'w') as file: file.write('Hello, World!') # Appending to a file with open('log.txt', 'a') as file: file.write('New log entry\n')
